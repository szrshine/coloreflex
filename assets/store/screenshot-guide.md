## ğŸ¨ Overlay Design Guidelines

### Text Overlay Specs
```
Primary Text (Main Message):
- Font: Bold, Sans-serif (Helvetica, Arial, Roboto)
- Size: 60-80pt
- Color: White with dark shadow, or dark with light glow
- Position: Top third of screen

Secondary Text (Subtitle):
- Font: Same family, regular or medium weight
- Size: 40-50pt
- Color: Match primary or complementary
- Position: Below primary text

Background:
- Semi-transparent gradient overlay
- Top portion darker for text readability
- Don't obscure gameplay
```

### Color Scheme
```
Primary Colors (from game):
- #4A90E2 (Blue)
- #9B59B6 (Purple)
- #E74C3C (Red)
- #F39C12 (Orange)
- #2ECC71 (Green)

Text Colors:
- White (#FFFFFF) with shadow
- Dark Blue (#1E3A5F) for contrast
- Gold (#FFD700) for highlights

Overlay Gradients:
- Linear gradient: rgba(0,0,0,0.5) to transparent
- Radial gradient: Center light, edges dark
```


### Recording Tools:
- **Mac**: QuickTime Player (Cmd+Shift+5)
- **Windows**: OBS Studio (free)
- **iOS Device**: Built-in screen recording
- **Android**: ADB screen recording

### Editing Tools:
- **iMovie** (Mac, free, simple)
- **DaVinci Resolve** (free, professional)
- **Adobe Premiere Rush** (mobile-friendly)
- **Kapwing** (online, easy)

### Video Specs:
```
Resolution: 1920x1080 (1080p) minimum
Format: MP4, H.264 codec
Frame rate: 30fps or 60fps
Duration: 15-30 seconds (max 2 min for Play Store)
Bitrate: 8-10 Mbps for quality
```

### Music:
- Use royalty-free music from:
  - YouTube Audio Library
  - Epidemic Sound (subscription)
  - Artlist (subscription)

- First 3 seconds are critical - hook immediately
- Show actual gameplay, not just graphics
- Keep text on screen long enough to read (2-3s)
- End with clear call-to-action

## ğŸ› ï¸ Quick Creation Scripts

### Take iOS Screenshot (Xcode Simulator)
```bash
# Open simulator
open -a Simulator

# Take screenshot (automatic save to Desktop)
xcrun simctl io booted screenshot screenshot.png

# Or use Cmd + S in Simulator window
```

### Take Android Screenshot (Emulator)
```bash
# List running emulators
adb devices

# Take screenshot
adb shell screencap -p /sdcard/screenshot.png

# Pull to computer
adb pull /sdcard/screenshot.png ~/Desktop/
```

### Batch Resize Images
```bash
# Install ImageMagick first: brew install imagemagick

# Resize to specific dimensions
magick convert input.png -resize 1290x2796 output.png

# Batch resize all PNGs in folder
for file in *.png; do
  magick convert "$file" -resize 1290x2796 "resized-$file"
done
```
